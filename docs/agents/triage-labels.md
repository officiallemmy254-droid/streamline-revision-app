# Triage Labels

This project uses a standard set of triage labels to manage the lifecycle of issues and tasks.

## Vocabulary

| Label | Description |
|-------|-------------|
| `needs-triage` | The issue is new and needs to be evaluated by a maintainer. |
| `needs-info` | More information is required from the reporter to proceed. |
| `ready-for-agent` | The issue is fully specified and can be implemented by an AI agent. |
| `ready-for-human` | The issue requires human intervention or manual implementation. |
| `wontfix` | The issue has been evaluated and will not be addressed. |
